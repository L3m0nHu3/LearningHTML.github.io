Steps in building an image
  • Pick an image (careful of the file extension in the URL)
  • Record the URL, cite it, comment on the date accessed, and recod the original, dimensions
  • Note: opening in a new tab will eventually provide a http or https URL with a file extension
  • Use blockquote and comment as appropirate
  • Use img-tag, with attributes src, alt, title, and style with width-height
  • Compare the same image with two sizes of the veiwport (25% & 100%)
